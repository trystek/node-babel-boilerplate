# Node-Babel boilerplate

Simple NodeJS boilerplate with Babel transpiler, Nodemon watcher and Jest testing.

## Commands

### Dev

```Shell
npm run dev
```

Rebuilding on changes in `src` directory.

### Build

```Shell
npm run build
```

Build into `dist` directory.

### Start

```Shell
npm start
```

Build and run.

### Test

```Shell
npm test
```

Build and run tests.
